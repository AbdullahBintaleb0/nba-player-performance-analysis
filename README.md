## Overview

This project applies machine learning techniques to analyze NBA player performance during the 2024–2025 season.

Using game-by-game player statistics, three core machine learning tasks were addressed:

* Regression to predict player scoring output.
* Clustering to identify player playing styles.
* Classification to predict game outcomes.

Additionally, temporal performance analysis was conducted to study player performance trends throughout the season.

## Dataset

The dataset contains NBA player statistics from the 2024–2025 season.

Two datasets were used:

### database_24_25.csv

Contains game-by-game player statistics where each row represents a player's performance in a single game.

### aggregated_player_stats.csv

Contains season-level player statistics generated by aggregating player performance across all games.

---

## Features

The dataset includes:

* Minutes Played (MP)
* Field Goals (FG, FGA, FG%)
* Three-Point Shooting (3P, 3PA, 3P%)
* Free Throws (FT, FTA, FT%)
* Offensive and Defensive Rebounds
* Assists (AST)
* Steals (STL)
* Blocks (BLK)
* Turnovers (TOV)
* Personal Fouls (PF)
* Points Scored (PTS)
* Game Score (GmSc)

---

## Feature Engineering

Several features were engineered to improve model performance:

* True Shooting Percentage (TS%)
* Points Per Minute (PPM)
* Assist-to-Turnover Ratio

These features provide better representations of player efficiency and effectiveness.

# Task 1: Player Performance Prediction (Regression)

The objective was to predict player points scored (PTS).

Models used:

* Support Vector Regression (SVR)
* Random Forest Regressor
* Multi-Layer Perceptron (MLP)

### Results

| Model         | RMSE | R² Score |
| ------------- | ---: | -------: |
| SVR           | 3.9% |  99.995% |
| MLP           | 5.7% |  99.995% |
| Random Forest | 9.5% |  99.987% |

SVR and MLP achieved the highest predictive performance.

---

# Task 2: Player Style Clustering

The goal was to identify player archetypes using season statistics.

Algorithms:

* K-Means Clustering
* Hierarchical Clustering

Player groups included:

* Superstar players
* Average players
* Developing players

Clustering helped identify playing styles and statistical similarities.

---

# Task 3: Game Outcome Classification

The objective was to predict whether a player's team won or lost.

Models:

* Support Vector Machine
* Decision Tree
* Logistic Regression

### Classification Results

| Model               | Accuracy | F1-Score | ROC-AUC |
| ------------------- | -------: | -------: | ------: |
| SVM                 |    57.5% |    57.6% |   57.6% |
| Logistic Regression |    57.3% |    61.4% |   56.7% |
| Decision Tree       |    47.9% |    60.4% |   47.8% |

SVM achieved the best overall performance.

---

# Temporal Analysis

The project also analyzed player performance trends throughout the season using rolling averages and efficiency metrics.

This analysis identified players with improving or declining performance over time.
